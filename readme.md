# yus

* https://github.com/zdavatz/yus

## DESCRIPTION:

ywesee user server. Works with the ywesee webframework and all the ywesee
software packages.

## FEATURES/PROBLEMS:

* Only runs on Ruby >= 2.1.0

## INSTALL:

* sudo gem install yus
